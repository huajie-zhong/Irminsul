# Workflows

Cross-component narratives — how a request flows through the system, how a user account is created, how data ingest happens end to end.

## Scope & Limitations

Only the check pipeline flow is currently documented. Init, render, and regen flows are not yet written. This layer does not cover configuration authoring or CI integration — those are in [`00-foundation`](../00-foundation/enforcement.md) and [`60-operations`](../60-operations/INDEX.md).
